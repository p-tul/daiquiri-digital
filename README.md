## Blog Posts
Markdown files contained within **src/pages/articles/...**

Create a new folder for each post and store markdown files, images and any other assets within.

Frontmatter must contain the following:

```
---
title: ''
date: 'yyyy-mm-dd'
updated: 'yyyy-mm-dd'
author: 
published: boolean
path: '/'
featuredImage: './'
---
```
